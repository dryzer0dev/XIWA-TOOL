## Installation

1. Téléchargez le fichier ZIP de XiwA Tool
2. Extrayez le contenu du dossier
3. Lancez `Setup.bat`

Ou

1. Ouvrez un terminal
2. Exécutez `git clone https://github.com/votre-repo/XIWA-TOOL.git`
3. Exécutez `cd XIWA-TOOL`
4. Exécutez `git pull`
5. Exécutez `setup.bat`
